1. First install the laravel libraries and dependencies
2. change the DB_DATABASE location file path of exact of your file path
3. run the migrations using php artisan migrate
4. then need to run the super admin seeder using php artisan db:seed RoleSeeder
5. then the project will be ready use after running the php artisan serve
6. visit the route from console the project will be visible and in action
